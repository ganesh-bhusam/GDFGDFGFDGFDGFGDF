
# AdvScribbl - Recent Updates & Features

## 1. UI & Layout Improvements
* Reorganized the main layout: The Color Panel is now positioned perfectly to the right of the canvas, and the Chat Panel is below the canvas.
* Fixed the Shapes Menu: Increased the frame size so shapes are clearly visible instead of appearing as empty white circles.
* Adjusted the Premium Color Tools: Moved the color ring and the 'Random Color' button to the right side of the active color preview square for better UX.

## 2. Drawing Tool Fixes
* Fixed the Random Color Button: It now instantly selects a random color when clicked, instead of slowly transitioning through hues.
* Restored the Redo Button: Fixed functionality so players can redo undone drawing strokes.
* Adjusted brush size controls layout in the premium tools area.

## 3. Game Flow & Room Stability
* Fixed Game Freezes: Resolved issues where the game would freeze or get stuck when transitioning between rounds or when the game finishes.
* Fixed Word Selection: Resolved logic confusions during the word selection phase.
* Improved Room Management: If a drawer leaves mid-game, the turn now automatically shifts to the next player. The game gracefully waits and resumes when players join an empty or 1-player room.

## 4. AFK & Timer Systems
* AFK Kick Timer: Implemented a 35-second AFK timer. If a drawer doesn't touch the canvas within 35 seconds of their turn starting, they are automatically kicked from the room with a message.
* Fixed Frozen Clock: The visual timer clock now accurately counts down during Word Selection, Round Transitions, and the Game Over Podium (it previously only worked during the drawing phase).

## 5. Visual Enhancements & Polish
* Cleaned Up Podium Screen: The top header text is now properly cleared during the Game Over podium screen, preventing old words or "GUESS THE WORD" from lingering.
* Delta Score Display (+Points): During the 5-second round reveal, the exact points a player earned (e.g., +788) are now visually displayed next to their total score in the player list.
* Better Guessed Highlights: The green highlight on a player's card (indicating they guessed correctly) is now brighter and more visible. It also correctly resets at the very start of every new drawer's turn.

## 6. Brand New Features
* Live-Stream Floating Emojis: Added a bar of high-quality emojis below the chat input. When clicked, emojis spawn and float up across the drawing canvas (similar to TikTok/Instagram live streams).
* Emoji Anti-Spam: Implemented a server-enforced limit where players can only send a maximum of 5 emojis per drawing. The limit automatically resets for everyone when a new drawing starts.

## 7. Recent UI & Chat Polish (2026-07-03 17:01)
* Player Panel Speech Bubbles: When a player types in chat or sends an emoji, a dynamic speech bubble now temporarily pops out from the right side of their player card.
* Increased Emoji Limit: Raised the floating emoji click limit from 5 to 15 per drawing round.
* Reversed Chat Message Order: Changed the chat box to display the newest messages visually at the top (using CSS column-reverse), making it easier to track recent guesses without looking down.
* Guesser Chat Scrolling Fix: Fixed an issue where the chat panel would expand infinitely downwards when the user was guessing. The chat grid row is now strictly capped, and the internal panel strictly handles scrolling correctly.
* Chat Scrollbar Aesthetics: Updated the chat scrollbar color to a sleek, semi-transparent black for much better contrast against the white chat background.

## 8. Database & Architecture Overhaul (2026-07-03)
* Replaced JSON flat-files with a robust node:sqlite database, secured the Git repository via .gitignore, consolidated the frontend to be served directly from the backend, and securely rotated production credentials (JWT and Razorpay).

## 9. Security, Architecture & Stability (2026-07-03)
* Ignored sensitive files from version control: Added `.gitignore` to both frontend and backend to protect `.env` secrets and JSON database files.
* Architecture shift: Updated the Express server to statically serve the frontend assets so both frontend and backend run on the exact same port and origin, solving CORS and deployment complexity.
* Cleaned up dead SQLite files (`skribbl.db`, etc.) left over from a previous database iteration.
* Fixed Backend Crash Bug: Patched a race condition in `gameEngine.js` where `scheduleHints` was not properly clearing its interval on round transitions, leading to a `TypeError` when reading `currentWord.length` and crashing the server.
* Expanded Word Banks: Dynamically merged a new, larger multi-language word bank setup, increasing the English vocabulary to nearly 3700 words and adding full support for German, Spanish, French and other regional variations.

## 10. Recent UI Layout & Styling Fixes (2026-07-06)
* Home Avatar Frame: Centered the avatar inside the frame and increased its display size for better visibility.
* Game Panel Spacing: Added margins to `.panel-title` and `.section-label` to prevent overlapping with decorative screws.
* Footer Info Pages (About, Privacy, Contact): Completely overhauled the layout. Removed the grid constraint so pages are wide and centered, applied a transparent glass panel effect with `#fff8e7` text, and added the doodle background. Home page links use a crisp `#1B5E20` and `#77625e` color scheme.
* Home Header Polish: Styled the Welcome Text, Unlock Premium button, and Logout button with dark brown colors so they contrast beautifully and are fully legible against the light wooden board.
* Word Selection UI: Modified the `Choose a word` overlay so that the word choice buttons properly stack vertically below the header text, eliminating UX lag and layout confusion.

## 11. Custom Mouse Cursor & Game Fixes (2026-07-16)
* Added a toggle button for the custom mouse cursor in the drawing tools and moved it to the game-bar so it is visible to all players.
* Fixed socket flooding by batching draw commands, and the chat is now properly cleared when re-joining.
* Fixed multiple bugs including: name validation, URL invite join logic, welcome text, and cursor toggle safety.
* Fixed invite link generation in the lobby and correctly wired up the copy button.

## 12. UI Layout Adjustments (2026-07-16)
* Fixed UI clipping of the player count badge and moved it to the left side of the Players panel header. Made panel headers extra bold.
* Added cache-busters to assets to force client updates and fixed panel title overlapping.

## 13. Dynamic Result Podiums & Transitions (2026-07-16)
* Added smooth transitions throughout the game.
* Implemented a new round end scoreboard.
* Added an interactive, dynamic final result podium at the end of the game.

## 14. Advanced Color Picker Integration (2026-07-16)
* Replaced the standard "Random Color" wheel button with a fully-featured, popup Advanced Color Picker.
* Integrates a visual color wheel, HSV/RGB sliders, alpha transparency, HEX input, an eyedropper tool, and dynamic palette generation.
* Color selections seamlessly tie into the active drawing tools without disruption.
